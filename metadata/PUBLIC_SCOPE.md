# Public release scope

This first public release contains the administrative geography needed by the
Sardinia One Health AMR Atlas web preview:

- municipalities (377 features);
- provinces (6 features);
- Sardinia region (1 feature).

The GeoJSON files are derived from the Regione Autonoma della Sardegna DBGT
10K V05 administrative layers and converted to WGS84 with Mapshaper.

This release does **not** publish point-level genomic data, farm coordinates,
clinical data, private datasets, or the partially acquired SIRA snapshot.
Those sources remain subject to the public/restricted model, licence checks,
aggregation and validation described in the project documentation.
