# Sardinia One Health AMR Atlas

Struttura del progetto:

- `public/`: dati destinati alla pubblicazione;
- `private/`: dati riservati, esclusi dal repository pubblico;
- `raw/`: copie immutabili delle fonti originali;
- `releases/`: pacchetti ZIP e release congelate;
- `reports/`: audit, controlli e report tecnici;
- `scripts/`: script di acquisizione e trasformazione;
- `metadata/`: fonti, vocabolari e manifest;
- `schema/`: schemi dati;
- `web/`: applicazione web/GitHub Pages.

La presenza di un file in `public/` non implica automaticamente che sia
pubblicabile: ogni dataset deve avere licenza, provenienza e stato di privacy
verificati.
